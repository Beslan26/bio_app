```
/auth
  POST /register
  POST /login
  POST /admin/create-doctor
  POST /doctor/login
  POST /refresh
  POST /password-recovery/request
  POST /password-recovery/confirm
  GET /me

/patient
  GET  /profile
  PUT  /profile
  POST /complaints/text
  POST /complaints/voice
  GET  /complaints
  GET  /tasks
  POST /tasks/{task_id}/entries
  GET  /tasks/{task_id}/chart

/doctor
  GET /profile
  GET /patients
  GET /patients/{id}
  GET /patients/{id}/complaints
  POST /tasks

/ai
  POST /extract-medical-facts
  POST /recommend-specialist

/notifications
  POST /send
```

выглядеть будеть примерно так:

```
app/
└── api/
    ├── main.py
    └── v1/
        ├── api.py           # Собирает все роутеры v1
        └── endpoints/
            ├── auth.py      # /register, /login, /doctor/login
            ├── patient.py   # /profile, /complaints, /tasks
            ├── doctor.py    # /profile, /patients, /tasks
            ├── ai.py        # /extract-medical-facts...
            └── notify.py    # /send

```
